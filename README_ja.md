## 🚀 ワンクリックインストール

iTrade は **Linux / macOS / Windows** 対応のワンクリックインストールスクリプトを提供しています。**システムアーキテクチャを自動検出**（AMD/ARM）、**GitHub から最新バージョンを自動取得**します。

<!-- TABS_START -->

### 🐧 Linux / 🍎 macOS

ターミナルを開き、以下のコマンドを実行すると、ダウンロード、インストール、起動がすべて完了します：

```bash
curl -fsSL https://raw.githubusercontent.com/itradeicu/itrade/main/install.sh | sudo bash
```

> 💡 **ヒント**：
> - `sudo` 権限が必要です（初回 Docker インストール時）
> - `x86_64` (AMD) と `aarch64/arm64` (ARM) アーキテクチャに自動対応
> - [GitHub Releases](https://github.com/itradeicu/itrade/releases) から最新バージョンを自動取得

### 🪟 Windows

**方法 1：PowerShell リモート実行（推奨）**

PowerShell を管理者として開き、以下を実行：

```powershell
irm https://raw.githubusercontent.com/itradeicu/itrade/main/install.bat | cmd
```

**方法 2：手動ダウンロード**

1. インストールスクリプトをダウンロード：[install.bat](https://github.com/itradeicu/itrade/releases/latest/download/install.bat)
2. 右クリック → **管理者として実行**

> 💡 **ヒント**：
> - 事前に [Docker Desktop](https://docs.docker.com/desktop/install/windows-install/) のインストールが必要
> - `AMD64` と `ARM64` アーキテクチャに自動対応

<!-- TABS_END -->

## 📖 プロジェクト紹介

iTrade はプロのトレーダーとクオンツチーム向けに設計されたエンタープライズグレードのクオンツ取引システムクライアントです。複数の成熟したクオンツ戦略、セキュリティ暗号化メカニズム、リアルタイム市場監視機能を統合し、110 以上の主要取引所の自動取引をサポートします。

## ⭐ コア機能モジュール

### 📈 グリッド取引 - レンジ相場で稼ぐ

価格が往復する時が一番儲かる。レンジ内で自動的に安く買って高く売る、手動操作不要。

- ✅ 現物と先物の両方に対応
- ✅ ロング、ショート、双方向のグリッド注文が可能
- ✅ マーチンゲール戦略：下落するほど買い増し、コストを下げる
- ✅ 4 つのリスク管理模式：利確・損切価格設定、收益率による自動停止

### 💰 資金調達率裁定 - 手数料で稼ぐ

パーペチュアル先物は 8 時間ごとに資金調達率を決済します。この機能はまさにそのためのものです。

- ✅ レートが正ならショート、負ならロング、手数料を取って即撤退
- ✅ タイミングは秒単位で逃さない
- ✅ 決済時に逆方向のポジションを絶対に開かない、安全性◎

### 📊 ボリュームメーカー - 取引量増加

特定の取引ペアを活発に見せたい？これを使って。

- ✅ 指値注文と成行注文の両方に対応
- ✅ 本物のトレーダー行動をシミュレート：注文金額ランダム、保有時間ランダム、休憩時間ランダム
- ✅ 目標取引量を設定、達成したら自動停止、無駄遣いなし

### 🔄 DCA（ドルコスト平均法）- 一括購入を避ける

一括で all-in しない。分割購入の方が安定。

- ✅ CEX と DEX の両方に対応
- ✅ 分、時間、日、月単位で投資、自由に設定
- ✅ ループ DCA：数秒ごとに購入、高頻度投資
- ✅ 最大回数を設定、達成したら自動停止

### 👀 相場モニタリング - 24 時間監視アシスタント

あなたが寝ている間も、モニターが見張っています。

- ✅ 複数の取引所を同時に監視
- ✅ オンチェーンアドレスの変化も監視可能
- ✅ DingTalk、Telegram、Discord、Webhook で通知

### 🔔 TradingView Webhook - シグナル自動取引

TradingView がシグナルを送り、システムが自動的に売買。

- ✅ 現物と先物取引をサポート
- ✅ セキュリティコード検証で偽造リクエストを防止
- ✅ 110 以上の取引所の Webhook 統合をサポート
- ✅ 全リクエストの完全なログ追跡

**使用方法**：
1. システム設定ページで Webhook トークンを作成
2. TradingView Alert 設定にトークンを追加
3. TradingView がシグナルをトリガーすると、システムが自動的に取引を実行

### 🔍 コイン分析 - スマートコインセレクター

どのコインを選ぶか分からない？筛选をお任せ。

**CEX コインスクリーニング**：
- ✅ 全市場の 24 時間データをスキャン
- ✅ 取引量、変動率、時価総額でフィルタリング
- ✅ 多次元ソートをサポート
- ✅ リアルタイム更新、热点を逃さない

**DEX ホットコイン**：
- ✅ オンチェーンのホットコイン追跡
- ✅ マルチチェーンフィルタリング（ETH、BSC、SOL など）
- ✅ 新規コイン発見、いち早く乗れる
- ✅ 時価総額と時間範囲のカスタマイズ

### 📢 マルチチャネル通知 - メッセージ完全カバー

重要なイベントが発生した時、第一时间でお知らせ。

- ✅ Telegram 通知
- ✅ Discord 通知

## 🔐 セキュリティは安心

あなたの資金とキーは、あなたが思う以上に私たちが守ります：

### 🛡️ データ伝送暗号化
ECDH 鍵交換 + AES-256 暗号化、銀行レベル。定期的な自動キー切り替え。

### 🔑 API キー暗号化ストレージ
取引所のキーは絶対に平文保存せず、AES-256 で暗号化して保存。

また、HKDF 鍵派生技術を使用し、各シナリオで異なるサブキーを使用します。ユーザーごとに暗号結果が異なり、データが同じでも暗号結果は完全に異なります。

> 💡 **データベースが漏洩？攻撃者はあなたのキーを取得できません。**

### 🏠 データセキュリティ
すべてのデータはユーザー側に保存され、ユーザーの秘密鍵、API キーなどの機密情報を絶対にアップロードしません。

## 🔧 一般的な運用コマンド

インストール後、以下のコマンドでサービスを管理：

```bash
# 実行ログを確認
docker logs -f itrade-icu

# サービスを起動
docker compose up -d

# サービスを停止
docker compose down

# サービスを再起動
docker compose restart

# アンインストール（コンテナとファイルを削除）
docker compose down && cd .. && rm -rf iTrade-Docker-ARM-v1.0.4
```
